**Hardware-ID ermitteln:**
- Die ID wird direkt im Aktivierungsfenster angezeigt вҶ’ `рҹ“Ӣ Kopieren` klicken
- Alternativ: `NRW_VC_Keygen.exe` auf dem Zielrechner starten вҶ’ ID oben rechts ablesen
- Oder das folgende Skript auf dem Zielrechner ausfГјhren:

```python
# hwid.py вҖ“ Hardware-ID ermitteln
import hashlib, uuid, platform, subprocess

parts = [f"{uuid.getnode():012x}", platform.node().lower()]
try:
    r = subprocess.run(['wmic','diskdrive','get','SerialNumber'],
                       capture_output=True, text=True, timeout=4)
    lines = [l.strip() for l in r.stdout.splitlines() if l.strip()]
    if len(lines) > 1: parts.append(lines[1])
except: pass

hwid = hashlib.sha256("|".join(parts).encode()).hexdigest()[:12].upper()
print("Hardware-ID:", hwid)
input()
```

## рҹ”’ Sicherheit

### Passwort-Hashing

Alle PasswГ¶rter werden mit **PBKDF2-HMAC-SHA256** gehasht und gesalzen:

```python

```

Das Klartext-Passwort wird **niemals** gespeichert.

### Lizenz-Signierung

LizenzschlГјssel werden mit **HMAC-SHA256** signiert:

```

```

- Der Signing-Key ist im Quellcode verschleiert (auf mehrere Variablen verteilt)
- Ohne den korrekten Signing-Key kГ¶nnen keine gГјltigen SchlГјssel erstellt werden
- Jede Manipulation am SchlГјssel wird beim PrГјfen erkannt

### Hardware-Bindung

Der Hardware-Fingerabdruck wird aus folgenden Komponenten berechnet:

